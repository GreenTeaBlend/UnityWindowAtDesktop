Unityで作成したアプリをデスクトップ壁紙とアイコンの間で描画するプロジェクトです。


具体的な方法としては、WindowControllerスクリプトが実行されたウィンドウのハンドルを取得
→ "WorkerW"というウィンドウの子ウィンドウに設定する、というもの。
(参考にさせていただきました：https://github.com/HatsuneMiku3939/3939LiveWallpaer)

よって、ビルドしたアプリだけでなく、UnityEditor上でプレイする際にもデスクトップでの描画となります。


また、デスクトップ壁紙の状態で他のアプリをクリックするとキー操作、マウス操作ともに受け付けなくなりますので、ゲームに利用するためにはタスクトレイなどに常駐する他のアプリで操作する必要があると思います。
